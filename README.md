# Memopad
A Basic Text Editor I made to run in Linux. 

This is designed using Lazarus IDE.

Some notable features:
Light and Dark Themes.
Recent Files Menu.

Status Bar Functions:
  - Click the font name to change font.
  - Click and hold mouse while moving it up and down to change Zoom. Right click to reset.
  - Left click the modified Indicator to save a file.
  - Left click the file path to open A file.
  - Right Click the file path to copy to clipboard.
